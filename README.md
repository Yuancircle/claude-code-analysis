## 🎯 核心发现

### Harness 概念
> **Harness 是 Claude Code 的本地运行时外壳**，基于 Bun 运行时构建，用 React 和 Ink 驱动终端 UI，提供文件系统访问、Shell 执行、分层记忆和声明式扩展能力。

### TAOR 循环
**Think-Act-Observe-Repeat** - Claude Code 的核心执行循环：
1. **Think** - LLM 生成响应（可能包含工具调用）
2. **Act** - 执行工具调用（Bash、文件编辑等）
3. **Observe** - 收集工具执行结果
4. **Repeat** - 将结果反馈给 LLM，继续循环
